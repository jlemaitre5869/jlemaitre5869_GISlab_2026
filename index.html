GIS & Cartography Portfolio

This website is a showcase of the work I have done during the GIS lab. The point of this course is to develops my skills in GIS by making different types of maps using different programs, such as QGIS and ArcGIS. Before going on to the work I have produced we need to understand what GIScience and GIS. GIScience , or Geographic Information Science, is the study of geographic information and phenomena. It looks at how geographic information works and how people use it. An example, of geographic information is how we determine space or how people interact with geographical technology. To help us understand this geographic information we use tools such as GIS. GIS allows us to bring spatial data and organise it in a coordinate system so that people can understand the information. This helps governments and other organisation make more informed decisions. For example, GIS can help determine if deforestation in an area has worsened over time by making a vegetation index map.

Map Portfolio

Choropleth Map of SDG

This choropleth project was to make a map of one of the SDG indicators. The method used to do this was to download the UN’s data for an indicator and then use QGIS to make the map. The purpose of my two maps was to show the two indicators for the target 5.5.

Vector Data Mapping

The vector project was to make two different maps: an isochrone map and a density map. For the isochrone map I used Open Street Maps and QGIS. For, the density map I used the same method as before, but I also used the data I collected, with my lab group, from a survey of the area. The isochrone map’s purpose was to show if necessary amenities are within walking distance to residence. The density map was to show how men and women score differently the same area on certain characteristics.

Digital Elevation Map

The Digital Elevation Map project was to use an old map of a river and make a DEM of the modern day area. The method to make this map required finding a map online and then using the DSM and DTM plugin on QGIS to create a map of the same area and overlay the old map. The purpose of the map is to show the differences between the two maps and see how they have changed and for what reasons.

Vegetation Index Map

The Vegetation Index project was to make a NDVI map of one area at two different years. To make these maps I used Google Earth Engine and QGIS. The purpose of this map was to see how the vegetation in the area has changed over time due to deforestation.

Image Classification

The Image classification project was to make a supervised, unsupervised and NDVI classification map. To make these maps I used ArcGIS pro. The purpose of these maps is to see the land use of an area and if it is impacting the natural landscape.

Multicriteria Analysis

The multicriteria analysis project was to examine a research paper that used this method with GIS. The purpose of this project is to allow me to fully understand the method and it’s importance for decision making.
